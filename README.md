# Python Programlama ve Algoritma Temelleri

Bu proje, temel programlama mantığının ve algoritmik düşünce yapısının Python dili üzerinden kavranmasını amaçlamaktadır.

## 📌 Teknik Müfredat ve Uygulama Alanları

Depo içerisindeki çalışmalar aşağıdaki temel programlama konseptlerini kapsamaktadır:

- **Objeler ve Veri Yapıları:** Sayısal (Integer, Float), metinsel (String) ve koleksiyon tipi verilerin bellekteki temsili ve yönetimi.
- **Operatörler:** Aritmetik, karşılaştırma ve mantıksal operatörlerin algoritma kurgusundaki rolleri.
- **Koşul İfadeleri (Control Flow):** `if`, `elif` ve `else` blokları ile karar mekanizmalarının oluşturulması.
- **Döngü Yapıları (Iteration):** `for` ve `while` döngüleri ile tekrarlı işlemlerin ve veri işleme süreçlerinin optimizasyonu.
- **Fonksiyonlar:** Kodun modüler hale getirilmesi, parametre yönetimi ve "Don't Repeat Yourself" (DRY) prensibinin uygulanması.

# Python Programming and Algorithmic Foundations

This project aims to grasp core programming logic and algorithmic thinking through the Python language.

## 📌 Technical Curriculum and Application Areas

The studies in this repository cover the following fundamental programming concepts:

- **Objects and Data Structures:** Representation and management of numeric (Integer, Float), textual (String), and collection-type data in memory.
- **Operators:** The roles of arithmetic, comparison, and logical operators in constructing algorithms.
- **Control Flow:** Creating decision mechanisms using `if`, `elif`, and `else` blocks.
- **Iteration (Loops):** Optimization of repetitive tasks and data processing through `for` and `white` loops.
- **Functions:** Implementation of modular code, parameter management, and the "Don't Repeat Yourself" (DRY) principle.
